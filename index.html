Marslink

Heliocentric satellite constellation simulator linking Earth and Mars.

Workspaces

  • Configure — tune the constellation with sliders. Search, type exact values, save presets.
  • Deployment Report — fine-grained per-orbit breakdown of vehicles, propellant, and costs.
  • Full Run — sweep multiple scenarios and compare results.

Keyboard

  • L toggle link latency labels
  • M toggle link Mbps labels

Tips

  • Drag in the 3D view to orbit, scroll to zoom.
  • Spot the Tesla Roadster.
  • Object sizes are not to scale; distances and physics are.

Not affiliated with SpaceX.
