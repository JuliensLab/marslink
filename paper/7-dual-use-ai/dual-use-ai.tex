% 7.1 The Case for In-Space AI
%   - AI compute demand growing faster than terrestrial data center capacity.
%   - Marslink constellation has TWO structural advantages over dedicated
%     LEO AI satellite constellations:
%     (a) High-bandwidth inter-satellite laser links ALREADY EXIST. Distributed
%         AI workloads (training, inference, model parallelism) require massive
%         inter-node bandwidth for gradient sync, KV cache sharing, parameter
%         exchange. LEO AI sats must build this interconnect from scratch.
%         Marslink gets it for free.
%     (b) Communication revenue amortizes the satellite bus, power, and launch.
%         AI compute is an incremental addition.

% 7.2 Solar Panel Area: The Main Tradeoff
%   - Marslink orbits at 1.0--1.52 AU; average ~1.3 AU.
%   - Solar irradiance: 1,361 W/m^2 (LEO) vs ~805 W/m^2 (1.3 AU).
%   - At 30% panel efficiency:
%     500W compute: LEO ~1.2 m^2, Marslink ~2.1 m^2 (1.7x)
%     2,000W compute: LEO ~4.9 m^2, Marslink ~8.3 m^2 (1.7x)
%   - These are modest areas relative to satellite bus.
%   - INCLUDE FIGURE: bar chart comparing LEO vs Marslink panel area.

% 7.3 Inter-Satellite Bandwidth for AI
%   - The other key tradeoff: distributed AI needs high bandwidth between nodes.
%   - Marslink satellites already have multi-Gbps laser links to neighbors.
%   - LEO AI constellations must build this from scratch -- significant added
%     cost and complexity per satellite.
%   - Marslink AI interconnect is essentially free.

% 7.4 Economics
%   - 3,000 sats with AI accelerators = 3,000-node distributed compute cluster.
%   - Incremental cost: $0.5--1M/sat for chip + extra solar panel.
%   - Total: $1.5--3B on a $15--21B base = 10--15% incremental investment
%     opening a second, potentially larger revenue stream.
